Discord WebOS App: a lightweight web shortcut to access Discord directly from your LG TV.

Enable Developer Mode on your TV.

Connect your TV via webOS CLI (ares-device-add).

Install the prebuilt .ipk:

ares-install --device <your-device> com.discord.web2_1.0.0_all.ipk


The app will appear on the Home screen.

Click to launch Discord Web directly.
